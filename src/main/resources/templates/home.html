Тестовое задание "Институт 1.0"

Что сделано:

  • Средствами Java разработана модель по картинке с заданием. В качестве CRUD интерфейса исполь- зован Spring Boot + Spring Web Mvc + Bootstrap
  • Реализован класс LoadDatabase, который осуществляет загрузку демонстрационных данных (пока студенты и курсы)
  • Реализован класс ExchangeController с методами сериализации существующих объектов модели в Json (экспорт), а также десериализации содержимого Json файла в объекты Java (импорт) с созданием от- сутствующих / апдейтом существубщих. Методы экспота / импорта основаны на использовании средств фреймворка Jackson с враппером RootHolder.
  • Объекты Java сохраняются в базу по spring-jpa + hibernate, в качестве СУБД используется PostgreSQL 10.5.21
  • Реализованы пара методов CRUD REST сервиса курсов.

Что не успел, но планирую доделать:

  • Реализация экспорта / импорта объектов модели средствами CRUD REST сервиса.
  • Реализация методов CRUD REST сервиса по работе с объектами модели Student, CourseProgress, Profes- sor
  • Spring security
  • Доработка LoadDatabase в части заполнения Прохождения курса студентами, реализация CRUD методов в REST и WebMVC по работе с этими объектами.
  • Наведение "красоты", тесты (в задании не упоминается), рефакторинг шаблонов страниц HTML.
